## 💻 快速开始 <a name="getting-started"></a>

### 后端
```sh
go get
go mod tidy
go run main.go
```

### 前端
```sh
yarn
yarn dev
yarn build
```

前端构建后`./dist`放到 `main.go` 同级目录
